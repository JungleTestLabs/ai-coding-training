# AI Native 实战训战营 — 两天版课程规格

> 2026-05-11 | V3 → V3.1
> 从游戏化调整为适合高级专业人士的案例研讨+方案对比+同行交流模式。

## 核心变化（V3→V3.1）

| 维度 | V3 | V3.1 |
|------|-----|------|
| 互动方式 | 游戏化（积分/徽章/排行榜） | 案例研讨 + 方案对比 + 圆桌对话 |
| 下午实战 | 竞赛制 | 问题解决工作坊（对比方案+同行评审） |
| 小组活动 | 积分竞争 | 用不同方法解同一问题，对比优劣 |
| 尾声 | 颁奖 | 最佳实践总结 + 深度讨论 |

## 目标学员调整

- 终端、鸿蒙、计算等**核心项目的高级/资深开发者**
- 有 AI Coding 使用经验，想在真实项目中建立系统化方法论
- 反感"培训感"，期待深度技术交流和可复用资产

## 防疲劳设计（保留并增强）

| 原则 | V3.1 体现 |
|------|----------|
| **25min 注意力周期** | 保留 |
| **形式轮换** | 讲授(20min)→Live Demo(15min)→小组讨论(15min)→实践(30min)→同行分享(15min) |
| **认知冲突** | "没优化 vs 优化后"的对比冲击 |
| **自主性** | 下午 Workshop 可选择不同难度的项目；讨论议题由学员提出 |
| **尊重感** | 不讲基础，直接进入深度内容；讲师定位为"同行分享"而非"老师教学生" |

## 两天时间线总览（不变）

```
Day 1: 基础 + 实战入门
  上午 (9:30-12:00)    L1 AI Native 范式  +  L2 Agent/Claude架构
  下午 (14:30-18:00)   Workshop 环境搭建 + 迁移实操 + 方案对比研讨

Day 2: 全流程 + 真项目
  上午 (9:30-12:00)    L3 全流程推演 Live Coding  +  L4 实战案例+代码走读
  下午 (14:30-18:00)   工程最佳实践研讨 + 端到端项目 + 同行评审 + 总结
```

## 新增：高级人士培训方法论

| 方法 | 说明 | 应用场景 |
|------|------|---------|
| **案例研讨** | 真实项目深度剖析，分组用不同方案解同一问题 | L4 代码走读 + Workshop 2 |
| **方案对比** | 各组展示不同实现路径，分析 trade-off | Workshop 2/3 结尾 |
| **圆桌对话** | 专家对谈，开放提问，自由交流 | L5 工程最佳实践 |
| **同行评审** | 互相审查代码，专业交流 | Workshop 3 结尾 |
| **问题解决工作坊** | 带自己的问题来，现场探讨方案 | Workshop 3 可选 |
| **开放空间** | 自组织讨论，参与者自己设定议题 | Day 2 结尾 Q&A |
