# AI Coding 实战培训 — 一日课程设计

> 基于 specs/update1.md 重构。面向有 AI Coding 经验的内部开发者。
> 不讲 AI 基础，只讲能立刻用到安卓转鸿蒙项目中的深度内容。

## 项目目标

设计一份一日培训课程（6h + 可选 2h 实操），覆盖：
- L1 AI Native 开发范式 — SDD 规范驱动 + 三层分工模型
- L2 Agent/Claude 架构深度解密 — Skills 机制 + 源码解密
- L3 全流程 AI Native 推演 — 需求→设计→实现→验证
- L4 安卓转鸿蒙实战案例 — 真实项目实战解读
- L5 工程最佳实践 — 效率/质量/安全/成本
- L6 落地策略 + 总结 — 可操作资产，第二天就能用

## 核心差异化

区别于市面上所有"AI入门培训"：
- ❌ 不讲什么是LLM、Transformer基础、AI历史和发展
- ✅ 基于 Agent 框架、SDD 规范驱动开发流程和最佳实践
- ✅ 基于真实安卓转鸿蒙迁移项目的实战经验
- ✅ 每个模块有可操作输出物（Skill模板/CLAUDE.md/Prompt模板等）

## V2 主要变化（vs V1）

| 变化 | V1 | V2 |
|------|----|----|
| 学员定位 | 1年+经验 | 有 AI Coding 经验 |
| Prompt 工程 | 独立 80min 模块 | 融入训前自学 |
| 安卓转鸿蒙实战 | 散落在其他模块 | 独立 L4 模块 |
| 结尾 | 16:40 结束 | +2h 可选实操 |
| 验收标准 | 5项 | 4项（更聚焦） |

## 知识体系

```
训前（自学 1.5h） → Prompt基础 + LLM概念 + Claude Code安装
    ↓
L1 AI Native 开发范式           9:15-10:00  (45min)
L2 Agent/Claude架构深度解密     10:00-11:00 (60min)
L3 全流程AI Native推演          11:10-12:00 (50min)
L4 安卓转鸿蒙实战案例           13:30-14:20 (50min)
L5 工程最佳实践                14:30-15:30 (60min)
L6 落地策略 + 总结             15:30-16:00 (30min)
    ↓
可选加时实操                    16:00-18:00 (2h)
```
