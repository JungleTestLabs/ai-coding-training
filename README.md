# AI Coding 实战培训 — 一日课程设计

> 面向内部开发者，基于安卓转鸿蒙实战经验，不讲AI基础，只讲能立刻用在工作中的深度内容。

## 项目目标

设计一份6小时+的一日培训大纲，覆盖：
- AI Native开发范式变革
- Claude Code/OpenCode架构与实战
- Prompt工程（从Transformer到CRISPE）
- 全流程AI协同推演（需求→设计→实现→测试）
- 工程判断力（质量/安全/成本）
- 落地策略与评估

## 核心差异化

区别于市面上所有"AI入门培训"：
- ❌ 不讲什么是LLM、Transformer基础
- ❌ 不讲AI历史和发展
- ✅ 基于真实安卓转鸿蒙迁移项目的Token优化经验
- ✅ 基于真实Claude Code/OpenCode工作流
- ✅ 基于SDD规范驱动开发流程
- ✅ 每个模块有可操作输出物（检查表/决策流程/Prompt模板）
