# AI Coding 实战培训 — AI Native 训战营（2天版）

> 从 1 天升级到 2 天。面向终端、鸿蒙、计算等重点项目开发者。
> 上午授讲+演示（2 模块）。下午实战+展示（项目制）。
> 游戏化积分制 + 小组竞赛 + 防疲劳形式轮换。

## 两天概览

```
Day 1: 基础 + 实战入门
  上午 (9:30-12:00)    L1 AI Native 范式  +  L2 Agent/Claude架构
  下午 (14:30-18:00)   Workshop1 环境搭建 + Workshop2 迁移实操 + 小组展示

Day 2: 全流程 + 真项目
  上午 (9:30-12:00)    L3 全流程推演  +  L4 安卓转鸿蒙案例
  下午 (14:30-18:00)   L5 最佳实践游戏 + Workshop3 端到端项目 + 评审颁奖 + L6总结
```

## 核心差异化

- ❌ 不讲 LLM/Transformer/AI 历史
- ✅ 讲授 35% + Live Demo 20% + 实战 45%（总实战 ~5.5h）
- ✅ 全程游戏化积分制 + 小组竞赛 + 成就徽章
- ✅ 25min 注意力周期 + 7 种形式轮换防疲劳
- ✅ 每天下午成果展示，制造 peak-end 体验
- ✅ 基于真实安卓转鸿蒙项目的 Skill/CLAUDE.md/Prompt 模板

## V3 主要变化（vs V2）

| 变化 | V2（1天） | V3（2天） |
|------|---------|---------|
| 实战时间 | 0 | ~5.5h（占比 50%） |
| 讲授比例 | ~60% | ~35% |
| 防疲劳 | 无 | 25min切换 + 7种形式轮换 |
| 游戏化 | 部分 | 全程积分制 + 徽章 |
| 结尾 | 总结 | 展示+评审+颁奖 |

## 目标学员

- 终端、鸿蒙、计算等内部重点项目开发者
- 有 AI Coding 使用经验，想系统提升效率和质量的工程师
- 需要在真实项目中落地 AI 工作流的团队
