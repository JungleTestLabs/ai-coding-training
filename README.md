## V2 两天概览

```
训前（自学 1.5h）→ Prompt基础 + LLM概念 + Claude Code安装

Day 1: 范式 + 架构 + 实战入门
  上午 L1 AI Native范式 + L2 Agent/Claude架构 + Live演示
  下午 Workshop1(环境搭建) + Workshop2(迁移实操) + 方案对比

Day 2: 全流程推演 + 实战案例 + 最佳实践
  上午 L3 全流程推演 Live Coding + L4 安卓转鸿蒙实战案例
  下午 Workshop3(端到端项目) + L5 工程最佳实践 + L6 落地策略
```

## L1-L6 知识体系

| 模块 | 内容 | 时长 |
|------|------|:--:|
| L1 | AI Native 开发范式 — SDD + 三层分工 | 60min |
| L2 | Agent/Claude架构 — 上下文/Skills/记忆/工具 | 75min |
| L3 | 全流程AI Native推演 — 需求→代码 Live Coding | 120min |
| L4 | 安卓转鸿蒙实战案例 — Skill机制+迁移铁律 | 50min |
| L5 | 工程最佳实践 — 效率/质量/安全/成本 | 50min |
| L6 | 落地策略 — 团队推行 + Token优化 + 总结 | 40min |
