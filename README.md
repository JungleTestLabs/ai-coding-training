# AI Coding 实战培训

> 面向终端、鸿蒙、计算等重点项目的高级开发者。
> 不讲AI基础，只讲能立刻用到安卓转鸿蒙项目中的实战内容。

## 课程版本

| 版本 | 文件 | 时长 | 说明 |
|------|------|:--:|------|
| **V2** | [00-课程大纲.md](docs/00-课程大纲.md) | 2天 | 6模块 + Live演示 + 3个Workshop实战 |
| V1 | [00-课程大纲-V1.md](docs/00-课程大纲-V1.md) | 1天 | 6模块纯讲授原版 |

## V2 两天概览

```
Day 1: 基础 + 实战入门
  上午 模块一(AI Native范式) + 模块二(Agent架构) + Live演示
  下午 Workshop1(环境搭建) + Workshop2(迁移实操) + 方案对比

Day 2: 全流程 + 真项目
  上午 模块三(Prompt工程) + 模块四(全流程推演) + Live Coding
  下午 Workshop3(端到端项目) + 模块五(工程判断力) + 模块六(落地策略)
```

## V2 vs V1

V2 是 V1 的演示和实战扩展版。6 个模块的教学内容不变，扩展了：

| 变化 | V1 | V2 |
|------|-----|-----|
| 时长 | 1天 (6h) | 2天 (~11h) |
| 演示 | 无 | 每模块增加 15-30min Live Demo |
| 实战 | 课内小练习 | 3个独立 Workshop (下午) |
| 交付物 | 7项 | 9项 (+CLAUDE.md+Skill模板) |
