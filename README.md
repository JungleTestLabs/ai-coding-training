# AI Coding 实战培训 — 两天版（V2）

> V1 扩展版。6 模块核心内容不变，增加演示和实战。
> 上午 9:30-12:00 讲授+演示 / 下午 14:30-18:00 实战+研讨。
> 面向终端、鸿蒙、计算等重点项目的高级开发者。

## 两天概览

```
Day 1: 基础 + 实战入门
  上午 模块一(AI Native范式) + 模块二(Agent架构) + Live演示
  下午 Workshop1(环境搭建) + Workshop2(迁移实操) + 方案对比

Day 2: 全流程 + 真项目
  上午 模块三(Prompt工程) + 模块四(全流程推演) + Live Coding
  下午 Workshop3(端到端项目) + 模块五(工程判断力) + 模块六(落地策略)
```

## 与 V1 的关系

V2 是 V1 的**演示和实战扩展版**。6 个模块的教学内容完全不变，仅在以下方面扩展：

| 变化 | V1 | V2 |
|------|-----|-----|
| 时长 | 1天 (6h) | 2天 (~11h) |
| 演示 | 无 | 每个模块增加 15-30min Live Demo |
| 实战 | 课内小练习 | 3个独立 Workshop (下午) |
| 交付物 | 7项 | 7项 + CLAUDE.md模板 + Skill模板 |
