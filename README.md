## V2 两天概览

```
Day 1: 范式 + 架构 + 推演
  上午 讲座(9:30-12:00) L1 AI Native范式 + L2 Agent/Claude架构
  下午 演示+演练 L1+L2 演示 → Workshop1 练手 → L3 Live Coding → 回顾
```
