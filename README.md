# AI Coding 实战培训 — AI Native 训战营（2天版）

> 面向终端、鸿蒙、计算等核心项目的高级/资深开发者。
> 上午讲授+演示。下午案例研讨+实战+方案对比。
> 不讲基础。案例驱动。同行交流。

## 两天概览

```
Day 1: 基础 + 实战入门
  上午 (9:30-12:00)    L1 AI Native 范式  +  L2 Agent/Claude架构
  下午 (14:30-18:00)   Workshop 环境搭建 + 多策略迁移实战 + 方案对比研讨

Day 2: 全流程 + 真项目
  上午 (9:30-12:00)    L3 全流程推演 Live Coding  +  L4 实战案例+代码走读
  下午 (14:30-18:00)   工程最佳实践圆桌 + 端到端项目 + 同行评审 + 总结
```

## 核心差异化

- ❌ 不讲 LLM/Transformer/AI 历史
- ❌ 不游戏化（无积分/徽章/排行榜）— 适配高级专业人士
- ✅ 案例驱动：真实项目全程推演 + 多策略对比分析
- ✅ 圆桌对话：讲师抛话题，学员主导讨论
- ✅ 同行评审：专业交流代替单向讲授
- ✅ 可带走资产：Skill模板/CLAUDE.md/Prompt集/审查清单
