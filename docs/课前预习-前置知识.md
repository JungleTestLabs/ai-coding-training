# 课前预习 — 前置理论知识（V3·2天版）

> 培训已升级为 2 天版，每天上午讲授+演示，下午实战+展示。
> 训前自学时间不变（~1.5h），但增加了实战环境的准备要求。

---

## 一、训前要求（与 V2 相同）

### 🔴 必修 1：LLM 核心概念（30min）
- Token、上下文窗口、System Prompt、Temperature、Prompt Caching
- 详见 V2 版预习文档核心概念表

### 🔴 必修 2：Claude Code 安装（15min）
```bash
npm install -g @anthropic-ai/claude-code
export ANTHROPIC_API_KEY=你的key
export CLAUDE_CODE_ENABLE_THINKING=false
export CLAUDE_CODE_AUTO_COMPACT=true
claude --version
```

### 🟡 推荐：Prompt Engineering 基础框架（30min）
- Zero-Shot / Few-Shot / Chain-of-Thought / Role Prompting

## 二、V3 新增：实战环境准备

由于 V3 每天下午都是实战，来之前请确保：

```bash
# 1. 验证 Claude Code 可用
claude "hello world"

# 2. 拉取实战用的演示项目
git clone https://github.com/JungleTestLabs/opencalc-harmonyos.git ~/training/opencalc-harmonyos

# 3. 确保 Git + GitHub CLI 可用
git --version && gh --version

# 4. 确保 Node.js >= 18
node -v
```
