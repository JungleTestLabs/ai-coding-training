|------|------|
| 目标学员 | 有 AI Coding 开发经验，想用 AI 提升效率的开发者 |

## 三、各模块详细设计

### Day 1 上午 — 讲座

#### L1: AI Native 开发范式 (60min) | 09:45-10:45

**目标**: 建立AI Native开发的正确心智模型。

| 时间 | 环节 | 内容 |
|:---:|------|------|
| 5min | 问题抛出 | "你上次用AI写代码踩了什么坑？" — 收集痛点贴便签 |
| 15min | 三层模型 | 传统开发 → AI增强 → AI Native。三角色模型 |
| 10min | SDD融合 | Spec-First → Agent执行 → 4项验收 |
| 15min | 实战数据 | 12模块迁移，Token从$1,440降到$3.50 |
| 10min | CRISPE速讲 | Prompt从Level 0→Level 3进化 |
| 5min | 小结 | "你不是在写代码，你是在给AI写需求规格书" |

---

#### ☕ 休息 (15min)

#### L2: Agent/Claude架构解读 (60min) | 11:00-12:00

**目标**: 理解Agent工作机制，掌握Token消耗和上下文管理。

| 时间 | 环节 | 内容 |
|:---:|------|------|
| 10min | 架构全景 | Agent = 上下文管理 + Skills + 记忆 + 驾驭 |
| 15min | Token消耗 | 系统提示词20-30K/次、文件读取30-50%、/compact |
| 10min | Skills机制 | 自包含/分层组合/CI验证/多平台适配 |
| 15min | 选型指南 | Claude Code vs OpenCode vs 混合方案 |
| 10min | 案例数据 | 12模块Token优化：$1,440→$120→$12→$3.50 |

### Day 2 上午 — 讲座+演练

#### L4: 增量开发与重构 (75min) | 09:30-10:45

**目标**: 掌握用 AI 做增量开发和代码重构的系统化方法论。

| 时间 | 环节 | 内容 |
|:---:|------|------|
| 10min | 场景引入 | 增量开发 vs 从零构建 — 不同场景 AI 策略差异 |
| 20min | AID增量流程 | Spec-First→原子任务→分支开发→编译验证→合并 |
| 20min | 重构实战 | AI辅助识别坏味道→安全重构(先测后改) |
| 15min | 反面教材 | 一次改太多/不做回归/信任AI重命名 |
| 10min | 经验总结 | 可复用的增量开发模式 + 重构决策框架 |

**关键教学点**：
- 增量铁律：Spec-First、每增量≤200行、先测后重构、独立可回滚
- 重构三原则：先写测试→一次一种→不通过就回滚
